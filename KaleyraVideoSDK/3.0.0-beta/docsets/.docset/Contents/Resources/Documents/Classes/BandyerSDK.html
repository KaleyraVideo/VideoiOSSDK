Logging

  • logLevel

    This property sets the logging level for the module. If you want to turn on logging facilities you can set this value to one of the logging levels specified BDKLogLevel enum (e.g. BDKLogLevelError, BDKLogLevelWarning and so on…). If you want to turn off logging you can pass BDKLogLevelOff value. The default value for this flag is BDKLogLevelOff.

Initialization

  • -configure:

    Configure the module with the configuration object provided as arguments.

    Once configured, the module cannot be reconfigured anymore.

    Throws an exception if this method is called more than once during the lifetime of the application. Throws an NSInvalidArgumentException exception if a blank appID argument is provided inside the config object. Throws an NSInvalidArgumentException exception if a nil config object is provided.
